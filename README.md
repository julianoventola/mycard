## Description

A simple flutter app to display a presentation card.

# Features

- Home page with personal card
- Challenge design (Challenge folder - not displayed)

## Getting Started

- Clone the repository
- run in cmd:
  ```cmd
  flutter pub get
  flutter run
  ```
